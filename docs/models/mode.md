# Mode

Mode for the fine-tuned model. Allowed values are 'general', 'character', 'style', 'product'. This will affect the caption behaviour. General will describe the image in full detail.


## Values

| Name        | Value       |
| ----------- | ----------- |
| `GENERAL`   | general     |
| `CHARACTER` | character   |
| `STYLE`     | style       |
| `PRODUCT`   | product     |
